1. Есть ли разница между $("h1 div"); и $("h1").$("div");
- может ли привести к тому что, поиск найдёт разные элементы?
Если может - приведите пример, когда.

При поиске по селектору $("h1 div") два параметра перечислены через пробел,
следовательно мы найдём первый элемент, который имеет сразу оба этих параметра.

При поиске по селектору $("h1").$("div") вначале мы найдем первый элемент
с параметром "h1", а затем среди его дочерних элементов выберем первый
с параметром "div".

Результаты поиска могут быть различными.
